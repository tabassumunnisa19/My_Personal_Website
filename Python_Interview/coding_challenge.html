Data types in python

Python provides built-in types like int, float, and complex. We can use type() function to identify the type of any number.

Example: 

num1 = 10
num2 = 3.14
num3 = 2 + 3j

print(type(num1))  # <class 'int'>
print(type(num2))  # <class 'float'>
print(type(num3))  # <class 'complex'>

Python supports tuple unpacking which allows swapping in a single line without an extra variable.

Example: 

a = 5
b = 10

a, b = b, a   # swap

print(a)   # 10
print(b)   # 5

We can iterate over each character and use isalpha(), isdigit(), and isalnum() checks to classify.

Example: 

text = "Hello123!@#"
letters = digits = specials = 0

for ch in text:
    if ch.isalpha():
        letters += 1
    elif ch.isdigit():
        digits += 1
    else:
        specials += 1

print("Letters:", letters)   # 5
print("Digits:", digits)     # 3
print("Specials:", specials) # 3

Python provides built-in functions max() and min() to find largest and smallest elements.

Example: 

numbers = [10, 3, 56, 23, 7]

largest = max(numbers)
smallest = min(numbers)

print("Largest:", largest)   # 56
print("Smallest:", smallest) # 3

We can reverse a string using a loop or reversed() function combined with join().

Example: 

text = "Python"
reversed_text = "".join(reversed(text))
print(reversed_text)   # nohtyP

Coding Challenges: Functions & Scope

Recursion is a technique where a function calls itself. Fibonacci series can be generated recursively by defining the base cases and recursive formula.

Example: 

def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Print first 7 Fibonacci numbers
for i in range(7):
    print(fibonacci(i), end=" ")   # 0 1 1 2 3 5 8

The *args allows a function to accept a variable number of positional arguments.

Example: 

def sum_numbers(*args):
    total = 0
    for num in args:
        total += num
    return total

print(sum_numbers(1, 2, 3, 4))   # 10
print(sum_numbers(5, 10))        # 15

The **kwargs allows a function to accept variable number of keyword arguments (dictionary format).

Example: 

def user_details(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

user_details(name="Alice", age=25, city="London")
# Output:
# name: Alice
# age: 25
# city: London

Factorial of a number n (n!) is the product of all positive integers less than or equal to n. Recursion makes it easy to implement.

Example: 

def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n-1)

print(factorial(5))   # 120

Global variables are defined outside functions, whereas local variables are defined inside functions. Python follows the LEGB rule (Local, Enclosing, Global, Built-in).

Example: 

x = 10   # global variable

def demo():
    x = 5  # local variable
    print("Inside function:", x)

demo()                 # Inside function: 5
print("Outside function:", x)   # Outside function: 10

Coding Challenges: Identifiers & Operators

Python identifiers must start with a letter or underscore and contain only letters, numbers, and underscores. The str.isidentifier() method helps check this.

Example: 

identifier = "my_var1"
print(identifier.isidentifier())   # True

identifier2 = "1var"
print(identifier2.isidentifier())  # False

Arithmetic operators perform basic mathematical operations like addition, subtraction, multiplication, division, modulus, exponentiation, and floor division.

Example: 

a = 10
b = 3
print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.3333
print(a % b)   # 1
print(a ** b)  # 1000
print(a // b)  # 3

Comparison operators compare two values and return a Boolean result (True or False).

Example: 

x = 5
y = 10
print(x == y)   # False
print(x != y)   # True
print(x > y)    # False
print(x < y)    # True
print(x >= 5)   # True
print(y <= 10)  # True

Logical operators combine Boolean expressions to return True or False.

Example: 

a = True
b = False
print(a and b)   # False
print(a or b)    # True
print(not a)     # False

The in operator checks if a value exists in a sequence, while not in checks the opposite.

Example: 

fruits = ["apple", "banana", "cherry"]
print("banana" in fruits)      # True
print("mango" not in fruits)   # True

Identity operators check whether two variables point to the same object in memory: is and is not.

Example: 

x = [1, 2, 3]
y = x
z = [1, 2, 3]

print(x is y)     # True (same object)
print(x is z)     # False (different objects)
print(x is not z) # True

def fibonacci(n):                        # Define function to print Fibonacci series
    a, b = 0, 1                          # First two numbers of Fibonacci sequence
    for _ in range(n):                   # Loop n times

        a, b = b, a + b                  # Update values for next iteration

def is_palindrome(s):                    # Define function to check palindrome

print(is_palindrome("madam"))            # True (palindrome)
print(is_palindrome("python"))           # False (not palindrome)

def remove_duplicates(lst):              # Define a function to remove duplicates
    result = []                          # Create an empty list to store unique elements
    for item in lst:                     # Iterate through each element in the list
        if item not in result:           # Check if item is not already in result
            result.append(item)          # Append unique item to result list
    return result                        # Return the final list without duplicates

nums = [1, 2, 2, 3, 4, 4, 5]             # Example list with duplicates
print(remove_duplicates(nums))           # Output ‚Üí [1, 2, 3, 4, 5]

def count_vowels_consonants(text):       # Function to count vowels & consonants
    vowels = "aeiouAEIOU"                # Define vowels
    v_count, c_count = 0, 0              # Initialize counters
    for char in text:                    # Loop through each character
        if char.isalpha():               # Check if character is a letter
            if char in vowels:           # Check if vowel
                v_count += 1
            else:                        # Otherwise consonant
                c_count += 1
    return v_count, c_count              # Return vowel & consonant counts

s = "Hello World"                        # Example string
print(count_vowels_consonants(s))        # Output ‚Üí (3, 7)
    

def is_anagram(s1, s2):                  # Function to check anagram
    return sorted(s1) == sorted(s2)      # Compare sorted strings

print(is_anagram("listen", "silent"))    # True (anagram)
print(is_anagram("hello", "world"))      # False (not anagram)

Object-Oriented Programming ‚Äì Coding Challenges
class Car:                                  # Define a Car class
    def __init__(self, brand, model):       # Constructor with brand and model
        self.brand = brand                  # Initialize brand attribute
        self.model = model                  # Initialize model attribute

    def display(self):                       # Method to display car info
        print(f"Car: {self.brand} {self.model}")

car1 = Car("Toyota", "Corolla")            # Create Car object
car1.display()                             # Call display method ‚Üí Car: Toyota Corolla
    
class Parent:                              # Parent class
    def greet(self):
        print("Hello from Parent")

class Child(Parent):                        # Child class inherits Parent
    def greet(self):                         # Override greet method
        print("Hello from Child")

c = Child()                                 # Create Child object
c.greet()                                   # Output ‚Üí Hello from Child
    
class A:
    def feature_a(self):
        print("Feature A")

class B:
    def feature_b(self):
        print("Feature B")

class C(A, B):                             # Multiple inheritance from A and B
    pass

obj = C()
obj.feature_a()                             # Output ‚Üí Feature A
obj.feature_b()                             # Output ‚Üí Feature B
    
class BankAccount:
    def __init__(self, balance):
        self.__balance = balance           # Private variable

    def get_balance(self):                # Getter method
        return self.__balance

    def set_balance(self, amount):        # Setter method
        if amount >= 0:
            self.__balance = amount
        else:
            print("Invalid amount")

acct = BankAccount(1000)
print(acct.get_balance())                  # Output ‚Üí 1000
acct.set_balance(2000)
print(acct.get_balance())                  # Output ‚Üí 2000
    
class Square:
    def __init__(self, side):
        self.side = side

    def area(self):
        return self.side ** 2              # Area of square

class Rectangle:
    def __init__(self, length, width):
        self.length = length
        self.width = width

    def area(self):
        return self.length * self.width    # Area of rectangle

shapes = [Square(4), Rectangle(3, 5)]     # List of different shapes

for shape in shapes:                       # Polymorphism in action
    print(shape.area())                     # Calls respective area() method
    
from abc import ABC, abstractmethod       # Import abstract base class

class Shape(ABC):                          # Abstract class
    @abstractmethod
    def area(self):                         # Abstract method
        pass

class Circle(Shape):                        # Concrete subclass
    def __init__(self, radius):
        self.radius = radius

    def area(self):                         # Implement abstract method
        return 3.14 * self.radius ** 2

c = Circle(5)
print(c.area())                             # Output ‚Üí 78.5

class Employee:
    def __init__(self, name, salary):
        self.name = name
        self.salary = salary

    def apply_increment(self):
        self.salary *= 1.1                   # Apply 10% increment

    def display(self):
        print(f"Employee: {self.name}, Salary: {self.salary:.2f}")

e1 = Employee("Alice", 50000)
e1.apply_increment()
e1.display()                               # Output ‚Üí Employee: Alice, Salary: 55000.00
