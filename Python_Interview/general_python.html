Python Interview Questions — Features & Data Types

Comprehensive questions and answers to help beginners understand Python features and data types.

Features & Data Types

Python is a high-level, interpreted programming language designed to be easy to read and write. Its main features include being interpreted, dynamically typed, high-level, object-oriented, simple, cross-platform, having extensive libraries, and supporting multiple programming paradigms.

Tip: Mention readability, simplicity, and versatility with examples if possible.

Python has numeric types (int, float, complex), sequence types (list, tuple, range), text type (str), set types (set, frozenset), mapping type (dict), boolean type (bool), and binary types (bytes, bytearray, memoryview).

Tip: Give examples of each for clarity.

is checks if two variables point to the same object in memory, while == checks if their values are equal. Example: a = [1,2]; b = [1,2]; a == b → True, a is b → False

Tip: Explain with a practical example.

List is mutable (can be changed), whereas tuple is immutable (cannot be changed). Tuples are faster and can be used as dictionary keys. Example: my_list = [1,2,3]; my_tuple = (1,2,3)

Tip: Highlight mutability and use cases.

A set is an unordered collection of unique elements, while a dictionary stores data in key-value pairs. Example: my_set = {1,2,3}; my_dict = {"a":1, "b":2}

Tip: Mention uniqueness vs mapping.

Shallow copy duplicates the top-level object but references nested objects. Deep copy duplicates all objects recursively. Example: import copy; shallow = copy.copy(obj); deep = copy.deepcopy(obj)

Tip: Use nested lists or dicts for demonstration.

Dynamic typing means you do not need to declare a variable’s type. Python determines it at runtime. Example: x = 10; x = "Hello"

Tip: Show flexibility but caution with type errors.

Python code is executed line by line, which allows quick testing and debugging. Unlike compiled languages, there is no separate compilation step.

Tip: Emphasize debugging advantages.

Python abstracts low-level details such as memory management and CPU instructions, allowing the programmer to focus on logic and algorithms.

Tip: Mention readability and ease of coding.

Python is easy to learn, has readable syntax, extensive libraries, multi-paradigm support, interpreted nature for quick testing, and is cross-platform. This makes development faster and beginner-friendly.

Functions like len(), type(), max(), min(), sum(), sorted() can be used with lists, tuples, sets, and dictionaries. Type conversion functions include str(), int(), float(), bool().

Mutable data types can be changed after creation (e.g., lists, dictionaries, sets). Immutable data types cannot be changed (e.g., tuples, strings, frozenset). This affects memory usage, performance, and dictionary key usability.

Functions & Scope

A function is a reusable block of code that performs a specific task. Functions help organize code, reduce repetition, and improve readability. They can take inputs (parameters) and return outputs (return values).

Tip: Give a simple example: def greet(name): return "Hello " + name

*args allows a function to accept any number of positional arguments, while **kwargs allows a function to accept any number of keyword arguments. They are useful when the number of inputs is unknown or flexible.

Tip: Example: 

def demo(*args, **kwargs):
    print(args)
    print(kwargs)
demo(1,2,3, a=10, b=20)

Recursion is when a function calls itself to solve a problem. It is used in tasks that can be broken down into smaller subproblems. A base case is required to avoid infinite recursion.

Example: 

def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n-1)

factorial(5)  # Output: 120

LEGB stands for Local → Enclosed → Global → Built-in, which is the order Python follows to resolve variable names. Python first looks in the local scope, then in enclosing functions (if nested), then global variables, and finally built-in names.

Tip: Understanding LEGB helps prevent variable shadowing and scope errors.

Positional arguments are passed in order, keyword arguments are passed with parameter names, and default arguments have predefined values if not provided. Example:

def greet(name, msg="Hello"):
    print(msg, name)

greet("Alice")          # Hello Alice
greet("Bob", msg="Hi")  # Hi Bob

Lambda functions are small anonymous functions defined using the lambda keyword. They are used for short, one-line operations and can take any number of arguments but return a single expression.

Example: square = lambda x: x*x; square(5) # 25

Common built-in functions include len(), type(), max(), min(), sum(), sorted(), enumerate(), any(), all(), zip(). These help manipulate data and iterate efficiently.

Tip: Provide small examples for each function.

Variable scope determines where a variable can be accessed. Local variables exist inside functions, global variables exist throughout the file, and nonlocal variables refer to enclosing function scopes. Using the global and nonlocal keywords, we can modify variables in outer scopes.

Example: 

x = 10
def func():
    global x
    x += 5
func()
print(x)  # 15

Functions make code reusable, modular, and easier to read. They allow breaking complex problems into smaller tasks, reduce duplication, and improve maintainability. Using functions also supports testing and debugging in isolation.

Identifiers & Operators

An identifier is a name used to identify a variable, function, class, module, or object in Python. It helps the interpreter recognize and reference the entity. Identifiers must start with a letter (A-Z, a-z) or underscore (_), followed by letters, digits, or underscores.

Example: my_var = 10 (valid identifier), 2var = 5 (invalid)

Rules for identifiers in Python:

  • Must begin with a letter or underscore (_).
  • Can contain letters, digits, and underscores.
  • Cannot be a reserved keyword.
  • Case-sensitive (e.g., Var and var are different).
  • No special characters or spaces allowed.

Example: Valid: myName, _count, Invalid: my-name, 123var

Operators are symbols or special characters used to perform operations on variables and values. Python has several types of operators to perform arithmetic, comparison, logical, bitwise, membership, and identity operations.

Arithmetic operators are used to perform basic mathematical operations like addition, subtraction, multiplication, division, modulus, exponentiation, and floor division.

Example: 

a = 10
b = 3
print(a + b)  # 13
print(a - b)  # 7
print(a * b)  # 30
print(a / b)  # 3.3333
print(a % b)  # 1
print(a ** b) # 1000
print(a // b) # 3

Comparison operators compare two values and return a Boolean result (True or False). Common comparison operators include:

  • == : equals
  • != : not equal
  • > : greater than
  • < : less than
  • >= : greater than or equal to
  • <= : less than or equal to

Example: 5 > 3 # True, 2 == 4 # False

Logical operators are used to combine conditional statements and return a Boolean value. They include:

  • and : True if both operands are True
  • or : True if at least one operand is True
  • not : Negates the Boolean value

Example: 

a = True
b = False
print(a and b)  # False
print(a or b)   # True
print(not a)    # False

Assignment operators assign values to variables. Common operators include:

  • = : assigns value
  • += : adds and assigns
  • -= : subtracts and assigns
  • *= : multiplies and assigns
  • /= : divides and assigns
  • %= : modulus and assigns
  • **= : exponent and assigns
  • //= : floor division and assigns

Example: 

a = 5
a += 3  # a = 8
a *= 2  # a = 16

Bitwise operators perform operations on binary representations of integers. They include:

  • & : AND
  • | : OR
  • ^ : XOR
  • ~ : NOT
  • << : Left shift
  • >> : Right shift

Example: 

a = 5   # 0101 in binary
b = 3   # 0011 in binary
print(a & b)  # 1
print(a | b)  # 7
print(a ^ b)  # 6
print(~a)     # -6

Membership operators check for existence of a value in a sequence like list, tuple, or string. They include:

  • in : True if value exists in sequence
  • not in : True if value does not exist in sequence

Example: 5 in [1,2,5] # True, 9 not in [1,2,5] # True

Identity operators compare whether two objects are the same in memory (object identity). They include:

  • is : True if both variables point to the same object
  • is not : True if variables point to different objects

Example: 

a = [1,2]
b = a
c = [1,2]
print(a is b)     # True
print(a is c)     # False
print(a == c)     # True

Conditional Statements & Loops (Flow Control)

Conditional statements are used to execute certain blocks of code only when specific conditions are met. They help in decision making within a program.

Python provides if, if-else, and if-elif-else statements for conditional execution.

Example: 

x = 10
if x > 5:
    print("x is greater than 5")
else:
    print("x is 5 or less")

  • if: Executes a block if a condition is True.
  • if-else: Executes one block if True, another if False.
  • if-elif-else: Checks multiple conditions in sequence, executes the first True block, else executes the final else block.

Example: 

marks = 75
if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
else:
    print("Grade C")

Loops are used to execute a block of code repeatedly until a condition is met or for a fixed number of times. Python has two main loops:

  • for loop: Iterates over a sequence (list, tuple, string, range, etc.).
  • while loop: Executes repeatedly while a condition is True.

Example (for loop): 

for i in range(5):
    print(i)  # Prints 0 to 4
Example (while loop): 
count = 0
while count < 5:
    print(count)
    count += 1

The break statement is used to **exit a loop prematurely**, even if the loop condition is still True. It stops the loop and moves the control to the next statement after the loop.

Example: 

for i in range(10):
    if i == 5:
        break   # Loop stops when i equals 5
    print(i)
# Output: 0 1 2 3 4

The continue statement **skips the current iteration** of a loop and moves to the next iteration without executing the remaining code in that iteration.

Example: 

for i in range(5):
    if i == 2:
        continue  # Skip printing 2
    print(i)
# Output: 0 1 3 4

The pass statement is a **placeholder**. It does nothing and is used when a statement is syntactically required but you don’t want to execute any code.

Example: 

for i in range(5):
    if i < 3:
        pass  # Do nothing for i=0,1,2
    else:
        print(i)
# Output: 3 4

Yes, Python allows **loops inside loops**. This is called nested loops. Useful for multidimensional data processing.

Example: 

for i in range(3):
    for j in range(2):
        print(i, j)
# Output:
# 0 0
# 0 1
# 1 0
# 1 1
# 2 0
# 2 1

Control flow statements (conditional statements and loops) are essential because they allow programs to make decisions and repeat actions. Without control flow, a program would execute statements sequentially and cannot handle dynamic scenarios.

Example: Choosing which discount to apply based on purchase amount or iterating over user inputs.
