Functions & Scope

A function is a reusable block of code that performs a specific task. Functions help organize code, reduce repetition, and improve readability. They can take inputs (parameters) and return outputs (return values).

Tip: Give a simple example: def greet(name): return "Hello " + name

*args allows a function to accept any number of positional arguments, while **kwargs allows a function to accept any number of keyword arguments. They are useful when the number of inputs is unknown or flexible.

Tip: Example: 

def demo(*args, **kwargs):
    print(args)
    print(kwargs)
demo(1,2,3, a=10, b=20)

Recursion is when a function calls itself to solve a problem. It is used in tasks that can be broken down into smaller subproblems. A base case is required to avoid infinite recursion.

Example: 

def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n-1)

factorial(5)  # Output: 120

LEGB stands for Local → Enclosed → Global → Built-in, which is the order Python follows to resolve variable names. Python first looks in the local scope, then in enclosing functions (if nested), then global variables, and finally built-in names.

Tip: Understanding LEGB helps prevent variable shadowing and scope errors.

Positional arguments are passed in order, keyword arguments are passed with parameter names, and default arguments have predefined values if not provided. Example:

def greet(name, msg="Hello"):
    print(msg, name)

greet("Alice")          # Hello Alice
greet("Bob", msg="Hi")  # Hi Bob

Lambda functions are small anonymous functions defined using the lambda keyword. They are used for short, one-line operations and can take any number of arguments but return a single expression.

Example: square = lambda x: x*x; square(5) # 25

Common built-in functions include len(), type(), max(), min(), sum(), sorted(), enumerate(), any(), all(), zip(). These help manipulate data and iterate efficiently.

Tip: Provide small examples for each function.

Variable scope determines where a variable can be accessed. Local variables exist inside functions, global variables exist throughout the file, and nonlocal variables refer to enclosing function scopes. Using the global and nonlocal keywords, we can modify variables in outer scopes.

Example: 

x = 10
def func():
    global x
    x += 5
func()
print(x)  # 15

Functions make code reusable, modular, and easier to read. They allow breaking complex problems into smaller tasks, reduce duplication, and improve maintainability. Using functions also supports testing and debugging in isolation.

Identifiers & Operators

An identifier is a name used to identify a variable, function, class, module, or object in Python. It helps the interpreter recognize and reference the entity. Identifiers must start with a letter (A-Z, a-z) or underscore (_), followed by letters, digits, or underscores.

Example: my_var = 10 (valid identifier), 2var = 5 (invalid)

Rules for identifiers in Python:

  • Must begin with a letter or underscore (_).
  • Can contain letters, digits, and underscores.
  • Cannot be a reserved keyword.
  • Case-sensitive (e.g., Var and var are different).
  • No special characters or spaces allowed.

Example: Valid: myName, _count, Invalid: my-name, 123var

Operators are symbols or special characters used to perform operations on variables and values. Python has several types of operators to perform arithmetic, comparison, logical, bitwise, membership, and identity operations.

Arithmetic operators are used to perform basic mathematical operations like addition, subtraction, multiplication, division, modulus, exponentiation, and floor division.

Example: 

a = 10
b = 3
print(a + b)  # 13
print(a - b)  # 7
print(a * b)  # 30
print(a / b)  # 3.3333
print(a % b)  # 1
print(a ** b) # 1000
print(a // b) # 3

Comparison operators compare two values and return a Boolean result (True or False). Common comparison operators include:

  • == : equals
  • != : not equal
  • > : greater than
  • < : less than
  • >= : greater than or equal to
  • <= : less than or equal to

Example: 5 > 3 # True, 2 == 4 # False

Logical operators are used to combine conditional statements and return a Boolean value. They include:

  • and : True if both operands are True
  • or : True if at least one operand is True
  • not : Negates the Boolean value

Example: 

a = True
b = False
print(a and b)  # False
print(a or b)   # True
print(not a)    # False

Assignment operators assign values to variables. Common operators include:

  • = : assigns value
  • += : adds and assigns
  • -= : subtracts and assigns
  • *= : multiplies and assigns
  • /= : divides and assigns
  • %= : modulus and assigns
  • **= : exponent and assigns
  • //= : floor division and assigns

Example: 

a = 5
a += 3  # a = 8
a *= 2  # a = 16

Bitwise operators perform operations on binary representations of integers. They include:

  • & : AND
  • | : OR
  • ^ : XOR
  • ~ : NOT
  • << : Left shift
  • >> : Right shift

Example: 

a = 5   # 0101 in binary
b = 3   # 0011 in binary
print(a & b)  # 1
print(a | b)  # 7
print(a ^ b)  # 6
print(~a)     # -6

Membership operators check for existence of a value in a sequence like list, tuple, or string. They include:

  • in : True if value exists in sequence
  • not in : True if value does not exist in sequence

Example: 5 in [1,2,5] # True, 9 not in [1,2,5] # True

Identity operators compare whether two objects are the same in memory (object identity). They include:

  • is : True if both variables point to the same object
  • is not : True if variables point to different objects

Example: 

a = [1,2]
b = a
c = [1,2]
print(a is b)     # True
print(a is c)     # False
print(a == c)     # True

Conditional Statements & Loops (Flow Control)

Conditional statements are used to execute certain blocks of code only when specific conditions are met. They help in decision making within a program.

Python provides if, if-else, and if-elif-else statements for conditional execution.

Example: 

x = 10
if x > 5:
    print("x is greater than 5")
else:
    print("x is 5 or less")

  • if: Executes a block if a condition is True.
  • if-else: Executes one block if True, another if False.
  • if-elif-else: Checks multiple conditions in sequence, executes the first True block, else executes the final else block.

Example: 

marks = 75
if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
else:
    print("Grade C")

Loops are used to execute a block of code repeatedly until a condition is met or for a fixed number of times. Python has two main loops:

  • for loop: Iterates over a sequence (list, tuple, string, range, etc.).
  • while loop: Executes repeatedly while a condition is True.

Example (for loop): 

for i in range(5):
    print(i)  # Prints 0 to 4
Example (while loop): 
count = 0
while count < 5:
    print(count)
    count += 1

The break statement is used to **exit a loop prematurely**, even if the loop condition is still True. It stops the loop and moves the control to the next statement after the loop.

Example: 

for i in range(10):
    if i == 5:
        break   # Loop stops when i equals 5
    print(i)
# Output: 0 1 2 3 4

The continue statement **skips the current iteration** of a loop and moves to the next iteration without executing the remaining code in that iteration.

Example: 

for i in range(5):
    if i == 2:
        continue  # Skip printing 2
    print(i)
# Output: 0 1 3 4

The pass statement is a **placeholder**. It does nothing and is used when a statement is syntactically required but you don’t want to execute any code.

Example: 

for i in range(5):
    if i < 3:
        pass  # Do nothing for i=0,1,2
    else:
        print(i)
# Output: 3 4

Yes, Python allows **loops inside loops**. This is called nested loops. Useful for multidimensional data processing.

Example: 

for i in range(3):
    for j in range(2):
        print(i, j)
# Output:
# 0 0
# 0 1
# 1 0
# 1 1
# 2 0
# 2 1

Control flow statements (conditional statements and loops) are essential because they allow programs to make decisions and repeat actions. Without control flow, a program would execute statements sequentially and cannot handle dynamic scenarios.

Example: Choosing which discount to apply based on purchase amount or iterating over user inputs.

Data Structure Methods & Functions (List, Tuple, Set, Dict, String & String Formatting)

A list is a mutable, ordered collection in Python that can store different data types. Mutable means we can change, add, or remove items after the list is created.

Example: 

fruits = ["apple", "banana", "cherry"]
fruits.append("orange")  
print(fruits)  
# Output: ['apple', 'banana', 'cherry', 'orange']

Some commonly used list methods:

  • append(x) – Add an item at the end.
  • insert(i, x) – Insert item at a specific position.
  • remove(x) – Remove the first occurrence of item.
  • pop(i) – Remove and return item at index (default last).
  • sort() – Sorts the list.
  • reverse() – Reverses the order of the list.

Example: 

numbers = [3, 1, 4, 2]
numbers.sort()
print(numbers)   # Output: [1, 2, 3, 4]

A tuple is an ordered, immutable collection in Python. Once created, you cannot modify its items.

Example: 

dimensions = (10, 20, 30)
print(dimensions[1])  
# Output: 20

Tuples are used when:

  • You want a collection of items that should not be changed (immutable).
  • They are faster than lists in iteration.
  • They can be used as dictionary keys (because they are hashable).

A set is an unordered collection of unique elements. Sets automatically remove duplicates.

Example: 

unique_nums = {1, 2, 2, 3, 4}
print(unique_nums)  
# Output: {1, 2, 3, 4}

  • union() – Combines elements from sets.
  • intersection() – Finds common elements.
  • difference() – Finds elements in one set but not another.

Example: 

A = {1, 2, 3}
B = {2, 3, 4}
print(A.union(B))         # {1, 2, 3, 4}
print(A.intersection(B))  # {2, 3}
print(A.difference(B))    # {1}

A dictionary is a collection of key-value pairs. Keys are unique and immutable, values can be of any type.

Example: 

student = {"name": "Alice", "age": 20}
print(student["name"])  
# Output: Alice

  • keys() – Returns all keys.
  • values() – Returns all values.
  • items() – Returns key-value pairs.
  • get(key) – Returns value for key (safe access).
  • update() – Updates dictionary with another dict.

Example: 

student = {"name": "Alice", "age": 20}
print(student.get("age"))    # 20
student.update({"grade": "A"})
print(student)               # {'name': 'Alice', 'age': 20, 'grade': 'A'}

A string is a sequence of characters enclosed in single quotes, double quotes, or triple quotes.

Example: 

message = "Hello World"
print(message[0])   # H

  • len(s) – Returns length.
  • s.lower() / s.upper() – Converts case.
  • s.strip() – Removes spaces.
  • s.replace(a, b) – Replaces substring.
  • s.split() – Splits into list.

Example: 

text = " Hello Python "
print(len(text))         # 14
print(text.strip())      # "Hello Python"
print(text.upper())      # " HELLO PYTHON "

String formatting allows inserting values into strings dynamically. Python provides several ways:

  1. Old Style (% formatting):
    2. name = "Alice"
print("Hello %s" % name)   # Hello Alice
  2. str.format() method:
    3. age = 25
print("I am {} years old".format(age))   # I am 25 years old
  3. F-strings (Python 3.6+):
    4. score = 95
print(f"My score is {score}")   # My score is 95
Regular Expressions (Regex) in Python

Regular Expressions (Regex) are patterns used to match, search, or manipulate strings. Python provides the re module to work with regex.

Example: Check if a string contains digits:

import re
pattern = r"\d+"   # matches one or more digits
text = "I have 2 cats"
match = re.search(pattern, text)
print(match.group())   # 2

Regex is used for:

  • Validating inputs (like emails, phone numbers, passwords)
  • Searching text inside documents or logs
  • Replacing or formatting strings
  • Extracting useful information from text
  • re.search() → finds the first match.
  • re.match() → checks for a match only at the start of the string.
  • re.findall() → returns all matches as a list.
  • re.finditer() → returns an iterator with match objects.
  • re.sub() → replaces matched patterns with new text.
  • re.split() → splits a string using regex.

Example:

import re
text = "My phone is 123-456-7890"
result = re.findall(r"\d+", text)
print(result)   # ['123', '456', '7890']
  • . → matches any character except newline
  • ^ → matches the start of the string
  • $ → matches the end of the string
  • * → 0 or more repetitions
  • + → 1 or more repetitions
  • ? → 0 or 1 repetition
  • {n} → exactly n repetitions
  • [] → matches characters inside brackets
  • \d → digit (0–9)
  • \w → word characters (a-z, A-Z, 0-9, _)
  • \s → whitespace

Example:

import re
pattern = r"[A-Za-z]+"
text = "Python 3 is awesome"
print(re.findall(pattern, text))  
# ['Python', 'is', 'awesome']

You can create a pattern that ensures the email has a valid structure.

Example:

import re
email = "test@example.com"
pattern = r"^[a-zA-Z0-9._%+-]+@[a-zA-Z0-9.-]+\.[A-Za-z]{2,}$"
if re.match(pattern, email):
    print("Valid Email")
else:
    print("Invalid Email")

Use re.sub() to replace matches with another string.

Example:

import re
text = "I love Java"
new_text = re.sub(r"Java", "Python", text)
print(new_text)   # I love Python
  • re.match() → checks only at the beginning of the string.
  • re.search() → scans the entire string for a match.

Example:

import re
print(re.match(r"cat", "catfish"))   # Match
print(re.search(r"cat", "fishcat"))  # Match
File Handling & Exceptions in Python

File handling in Python allows us to create, read, write, and manage files. It is useful for saving and retrieving data from files.

We use the built-in open() function for file operations.

f = open("sample.txt", "r")   # open for reading
print(f.read())
f.close()
  • "r" → Read (default, error if file does not exist)
  • "w" → Write (creates new or overwrites existing file)
  • "a" → Append (adds new data at the end of file)
  • "x" → Create (creates a file, error if exists)
  • "t" → Text mode (default)
  • "b" → Binary mode

We use read(), readline(), and readlines() to read data.

f = open("sample.txt", "r")
print(f.read())        # entire file
f.seek(0)
print(f.readline())    # one line
f.seek(0)
print(f.readlines())   # list of lines
f.close()

We use write() or writelines() to add content.

f = open("sample.txt", "w")
f.write("Hello, Python!")
f.close()

Now the file contains: Hello, Python!

Append mode ("a") adds new content at the end of the file without deleting existing content.

f = open("sample.txt", "a")
f.write("\nThis is appended text.")
f.close()

with open() is preferred because it automatically closes the file after use, even if an exception occurs.

with open("sample.txt", "r") as f:
    data = f.read()
    print(data)   # file auto-closed here

Exceptions are runtime errors that occur during execution. They can be handled to prevent program crashes.

Examples: ZeroDivisionError, FileNotFoundError, IndexError.

We use try-except blocks.

try:
    f = open("missing.txt", "r")
except FileNotFoundError:
    print("File not found!")

The finally block always runs, whether an exception occurs or not. It is useful for cleanup tasks like closing files.

try:
    f = open("sample.txt")
    print(f.read())
except FileNotFoundError:
    print("File not found")
finally:
    print("Execution finished")
  • Errors → problems in code syntax (cannot be handled at runtime).
  • Exceptions → runtime issues that can be handled with try-except.
  • ZeroDivisionError – division by zero
  • TypeError – operation on invalid data type
  • IndexError – accessing invalid list index
  • KeyError – accessing missing dictionary key
  • ValueError – invalid value for operation
  • FileNotFoundError – file does not exist
try:
    x = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")

We use the raise keyword to create our own exceptions.

age = -5
if age < 0:
    raise ValueError("Age cannot be negative!")
Iterators, Generators, Decorators, and Context Managers in Python

An iterator is an object that allows you to traverse through all the elements of a collection (like a list or tuple) one by one. It uses the methods iter() and next().

Example:

nums = [1, 2, 3]
it = iter(nums)

print(next(it))   # 1
print(next(it))   # 2
print(next(it))   # 3
  • Iterable → an object that can return an iterator (like list, tuple, set).
  • Iterator → the object that actually produces elements one at a time using next().

Example:

nums = [1, 2, 3]       # Iterable
it = iter(nums)        # Iterator
print(next(it))        # 1

A generator is a simpler way to create iterators using the yield keyword instead of returning all values at once. They are memory efficient.

Example:

def count_up_to(n):
    i = 1
    while i <= n:
        yield i
        i += 1

for num in count_up_to(3):
    print(num)

# Output: 1 2 3
  • return → ends the function and gives a value back.
  • yield → pauses the function, saves its state, and resumes later (used in generators).

Example:

def demo():
    yield 1
    yield 2
print(list(demo()))   # [1, 2]

A decorator is a special function that modifies the behavior of another function without changing its code. It uses the @ symbol.

Example:

def greet(func):
    def wrapper():
        print("Hello!")
        func()
        print("Goodbye!")
    return wrapper

@greet
def say_name():
    print("My name is Alice")

say_name()
# Output:
# Hello!
# My name is Alice
# Goodbye!

Decorators are useful for:

  • Logging
  • Authentication
  • Performance measurement
  • Access control

A context manager is used to manage resources like files or database connections. The most common way is using the with statement.

Example:

with open("example.txt", "w") as f:
    f.write("Hello World!")
# File is automatically closed after this block

You can create a custom context manager using the __enter__ and __exit__ methods or the contextlib module.

Example:

class MyContext:
    def __enter__(self):
        print("Enter context")
        return self
    def __exit__(self, exc_type, exc_value, traceback):
        print("Exit context")

with MyContext():
    print("Inside context")
Functional Programming (Lambda, map, filter, reduce, comprehensions)

Functional Programming is a programming paradigm that treats computation as the evaluation of mathematical functions and avoids changing-state and mutable data. In Python we use FP-style tools (functions as first-class citizens, lambdas, map/filter/reduce, comprehensions, generator expressions) to write concise, declarative code.

Why use FP?

  • Leads to shorter, clearer transformations on collections.
  • Pure functions (no side effects) are easier to test and reason about.
  • Generator expressions and lazy evaluation can improve memory usage.

Tip: FP is great for data pipelines (transform → filter → reduce). For complex logic prefer readable functions over clever one-liners.

A pure function always returns the same output for the same inputs and has no side effects (it does not modify external state, I/O, or its arguments). 

# Pure function example
def square(x):
    return x * x

# Impure function (side effect)
counter = 0
def inc():
    global counter
    counter += 1
    return counter

Tip: Favor pure functions for easier testing and parallelization.

A lambda creates a small anonymous function in one expression. Syntax: lambda args: expression. Use lambdas for simple, short functions passed to higher-order functions. 

# Lambda example
square = lambda x: x * x
print(square(5))  # 25

# Equivalent regular function
def square2(x):
    return x * x

Tip: Prefer named functions for complex logic — lambdas are for short inline usage only.

map(function, iterable) applies function to each item of iterable and returns an iterator (in Python 3). Convert to list() if you need the results immediately. 

# Double every element
nums = [1, 2, 3, 4]
doubled = map(lambda x: x * 2, nums)   # returns iterator
print(list(doubled))                    # [2, 4, 6, 8]

# Using a regular function
def double(x): return x * 2
print(list(map(double, nums)))          # [2, 4, 6, 8]

Tip: map is concise, but list comprehensions are often clearer in Python.

filter(function, iterable) returns those items of iterable for which function(item) is True. It returns an iterator. 

# Keep only even numbers
nums = [1,2,3,4,5,6]
evens = filter(lambda x: x % 2 == 0, nums)
print(list(evens))  # [2, 4, 6]

Tip: Use filter for boolean selection; list comprehensions often read better: [x for x in nums if x%2==0].

functools.reduce(function, iterable [, initializer]) cumulatively applies function to items of iterable, reducing it to a single value. Useful for sums, products, or other fold operations. 

from functools import reduce

nums = [1,2,3,4]
sum_all = reduce(lambda a, b: a + b, nums)   # 10
product = reduce(lambda a, b: a * b, nums)   # 24

# Note: use built-in sum(nums) for summation (clearer and faster)

Tip: Prefer built-ins (like sum()) when they exist; use reduce for non-standard folds.

Example: sum of squares of even numbers from 1..10

from functools import reduce

nums = range(1, 11)
even_squares = map(lambda x: x*x, filter(lambda x: x%2==0, nums))
result = reduce(lambda a, b: a + b, even_squares)
print(result)  # 220  (i.e. 2^2 + 4^2 + 6^2 + 8^2 + 10^2)

Tip: The same can be written more readably with a comprehension: sum(x*x for x in nums if x%2==0).

A list comprehension is a compact syntax for creating lists. It is usually more readable than chained map/filter calls in Python. 

# Using map/filter
nums = [1,2,3,4,5]
squared_evens = list(map(lambda x: x*x, filter(lambda x: x%2==0, nums)))

# Using list comprehension (preferred)
squared_evens2 = [x*x for x in nums if x%2==0]
print(squared_evens2)  # [4, 16]

Tip: Favor comprehensions for readability; use map/filter when you want to pass existing functions directly.

A generator expression looks like a list comprehension but uses parentheses and produces values lazily (one by one) — saving memory for large datasets. 

# List comprehension (creates full list)
squares = [x*x for x in range(1000000)]

# Generator expression (lazy, memory-friendly)
squares_gen = (x*x for x in range(1000000))
print(next(squares_gen))   # 0

Tip: Use generator expressions for pipelines and when working with large or infinite streams.

Use FP tools when: you have clear data transformations (map/filter/fold) and want concise, declarative code. Use generator expressions for memory efficiency.

Prefer loops or named functions when: the logic is complex, has many branches, or readability would suffer with nested lambdas.

Tip: Readability > cleverness. Write code your team can maintain.

Advanced Topics in Python

The Global Interpreter Lock (GIL) is a mutex in CPython (the standard Python interpreter) that allows only one thread to execute Python bytecode at a time.

Why does it exist?

  • CPython’s memory management is not thread-safe; GIL prevents race conditions.
  • Simplifies implementation of CPython and avoids certain low-level bugs.

Note: Because of GIL, Python threads cannot achieve true parallel CPU-bound execution. For I/O-bound tasks, threads still provide concurrency.

Python uses automatic memory management with reference counting and garbage collection to free unused memory.

  • Reference counting: Every object keeps a count of references pointing to it. When count reaches 0, memory is freed.
  • Garbage collector: Detects and cleans cyclic references that reference counting cannot free.
  • Memory pools: Python uses the pymalloc allocator for small objects to reduce system allocation overhead.

Example: Deleting objects manually or using del obj can reduce memory usage in long-running programs, but garbage collector usually handles it automatically.

Python provides both threads and processes to achieve concurrency. Their differences:

FeatureMultithreadingMultiprocessing
ExecutionRuns in same process, shares memory.Runs in separate processes, separate memory.
GIL ImpactLimited CPU-bound parallelism due to GIL, fine for I/O-bound tasks.No GIL restriction, true parallel execution possible.
MemoryShared memory space, less overhead.Separate memory space, higher overhead.
Use CasesI/O-bound tasks (file/network operations).CPU-bound tasks (heavy computation, data processing).

Example: Using threading for downloading files and multiprocessing for image processing in parallel.

Several ways to improve Python performance:

  • Use built-in functions and libraries (they are implemented in C, faster).
  • Use list/set/dict comprehensions instead of loops.
  • Avoid global variables inside loops (local variable lookup is faster).
  • Use generators for large data to save memory.
  • Use multiprocessing for CPU-bound tasks to bypass GIL.
  • Profile your code using cProfile or timeit to find bottlenecks.
  • Use third-party tools like NumPy, Cython, or PyPy for speed.

Example: Summing a large list: 

# Slow
sum = 0
for i in range(1000000):
    sum += i

# Faster
sum = sum(range(1000000))

Modules, Packages, Libraries, Import & pip in Python

A module is a single Python file (.py) that contains functions, classes, or variables which can be reused in other programs.

Example: 

# mymodule.py
def greet(name):
    return f"Hello, {name}"

# main.py
import mymodule
print(mymodule.greet("Alice"))   # Hello, Alice

A package is a collection of modules organized in directories with an __init__.py file. Packages help in structuring large projects.

Example: 

myproject/
│
├─ mypackage/
│   ├─ __init__.py
│   ├─ module1.py
│   └─ module2.py
└─ main.py

# main.py
from mypackage import module1
module1.function()

A library is a collection of packages and modules that provide pre-written code for common tasks. Python has built-in libraries and third-party libraries.

Example: math is a built-in library containing modules for mathematical operations: 

import math
print(math.sqrt(16))   # 4.0

The import statement allows you to bring modules or packages into your current program so you can reuse code without rewriting it.

  • Import entire module: import module_name
  • Import specific function: from module_name import func_name
  • Import with alias: import module_name as alias

Example: 

import math
print(math.factorial(5))   # 120

from math import factorial
print(factorial(5))       # 120

import math as m
print(m.sqrt(25))          # 5.0

pip is Python’s package installer. It is used to download and install third-party libraries and packages from the Python Package Index (PyPI).

  • Install a package: pip install package_name
  • Upgrade a package: pip install --upgrade package_name
  • Uninstall a package: pip uninstall package_name
  • List installed packages: pip list

Example: 

pip install requests   # Installs the 'requests' library for HTTP requests

After installing a library using pip, you can import its modules or functions using import statements.

Example: 

# Install library first
pip install requests

# Use in program
import requests
response = requests.get("https://api.github.com")
print(response.status_code)
