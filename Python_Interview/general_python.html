Functions & Scope

*args allows variable-length positional arguments; **kwargs allows variable-length keyword arguments.


def demo_args(*args, **kwargs):
    print(args)
    print(kwargs)

demo_args(1,2,3, name="Nisa", age=25)

Tip: Show example for flexibility in functions.


def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n-1)
print(factorial(5)) # 120

Python's scope resolution: Local → Enclosed → Global → Built-in.

Object-Oriented Programming (OOP)

Class: blueprint; Object: instance of class.


class Dog:
    def __init__(self, name):
        self.name = name

my_dog = Dog("Tommy")
print(my_dog.name) # Tommy

Self refers to the instance of the class and allows access to attributes and methods.

Special method called when an object is created. Used to initialize attributes.

Single, Multiple, Multi-level, Hierarchical. Allows reuse of code from parent class.

Public: accessible everywhere; Protected (_): accessible in class and subclasses; Private (__): accessible only inside class.

Same method name, different behavior in different classes (method overriding).

Hiding implementation using abstract base classes (ABC).

Regular Expressions

Regex allows pattern matching in strings. Python provides the re module for this.


import re

pattern = r'\d+'
text = "I have 2 apples and 3 bananas"
matches = re.findall(pattern, text)
print(matches)  # ['2', '3']

re.match(), re.search(), re.findall(), re.sub(), re.split()

Iterators, Generators, Decorators, Context Managers

An object with __iter__() and __next__() methods. Example:


my_list = [1,2,3]
it = iter(my_list)
print(next(it)) # 1
    

def gen_numbers(n):
    for i in range(n):
        yield i

for num in gen_numbers(5):
    print(num)
    

def decorator(func):
    def wrapper():
        print("Before function")
        func()
        print("After function")
    return wrapper

@decorator
def say_hello():
    print("Hello!")

say_hello()
    

with open('file.txt', 'w') as f:
    f.write("Hello World")
# Automatically closes file
Functional Programming (Lambda, map, filter, reduce)

square = lambda x: x**2
print(square(5)) # 25
    

from functools import reduce

nums = [1,2,3,4,5]

squared = list(map(lambda x: x**2, nums))
evens = list(filter(lambda x: x%2==0, nums))
sum_all = reduce(lambda a,b: a+b, nums)

print(squared, evens, sum_all)
Advanced Topics

Global Interpreter Lock (GIL) allows only one thread to execute Python bytecode at a time. Limits multi-threading in CPU-bound tasks.

Automatic garbage collection, reference counting, memory pool (PyMalloc), and dynamic allocation.

Multithreading: threads share memory, limited by GIL. Multiprocessing: separate memory, true parallelism.

Use built-in functions, list comprehensions, generator expressions, avoid global variables, use C extensions or PyPy.

Numpy, Pandas, Matplotlib, Seaborn, Scikit-learn, TensorFlow, PyTorch, Requests, BeautifulSoup, Flask, Django
