Python is a high-level, interpreted, general-purpose programming language. It is widely used for web development, data science, artificial intelligence, automation, and more.

Key Features of Python:

  • Easy to read and write with simple syntax.
  • Interpreted language (no compilation needed).
  • Object-oriented and supports functional programming.
  • Extensive standard library and third-party modules.
  • Cross-platform compatibility.
  • Dynamic typing and memory management.

Example: Hello World in Python

# Print "Hello, World!" to the console
print("Hello, World!")  # Output: Hello, World!

Python Basics forms the foundation of programming in Python. In this topic we cover variables, built-in data types (detailed), type conversion, input/output and comments ‚Äî with clear examples and exercises (solutions included).

1. Variables

A variable is a name that refers to a value stored in memory. Use the = operator to assign values.

# Creating variables
name = "Nisa"
age = 25
height = 5.4

print(name)    # Output: Nisa
print(age)     # Output: 25
print(height)  # Output: 5.4

2. What are Python's built-in data types? (detailed)

Data type tells Python what kind of value is stored and what operations are allowed. Python is dynamically typed ‚Äî types are determined at runtime.

Numeric types

int ‚Äî whole numbers. float ‚Äî decimal numbers. complex ‚Äî complex numbers with imaginary parts.

# Numeric examples
x = 10        # int
y = 3.14      # float
z = 2 + 5j    # complex

print(type(x))  # <class 'int'>
print(type(y))  # <class 'float'>
print(type(z))  # <class 'complex'>
Text and sequence types

str ‚Äî text (immutable). Sequences are ordered collections: list (mutable), tuple (immutable), and range (immutable, used for iteration).

# Sequences
text = "Hello"               # str
fruits = ["apple", "banana"] # list
coords = (10, 20)            # tuple

print(text[0])    # H
fruits[1] = "mango"  # lists can change
#print(coords[0] = 5)  # error: tuples are immutable
Mapping type

dict ‚Äî key → value mapping. Use for records and fast lookup by key. Keys must be hashable (immutable).

# Dictionary example
person = {"name": "Nisa", "age": 25}
print(person["name"])   # Nisa
print(person.get("email"))  # None (safe access)
Set types

set ‚Äî unordered unique items. frozenset ‚Äî immutable set. Good for membership tests and removing duplicates.

# Set example
A = {1, 2, 3, 3}  # duplicates removed -> {1,2,3}
B = set([2,4])
print(A & B)  # intersection
print(A | B)  # union
Binary types

bytes, bytearray, memoryview ‚Äî used for raw binary data (files, network I/O).

b = b"hello"            # bytes literal
ba = bytearray(b"abc")  # mutable bytes
ba[0] = 120
print(b, ba)
Boolean and None

bool ‚Äî True or False. None ‚Äî absence of value (NoneType).

print(type(True))    # <class 'bool'>
print(True + 1)      # 2 (bools are subclass of int)

x = None
print(type(x))       # <class 'NoneType'>

3. Type conversion (casting)

Convert between types using int(), float(), str(), list(), tuple(), set(), dict(). Be careful: int("3.14") raises an error ‚Äî convert to float first.

# Conversions
s = "15"
print(int(s) + 5)   # 20
print(float("3.14") + 1)  # 4.14

# bad conversion -> ValueError
# int("3.14")  # ValueError: invalid literal

4. Input & Output

input() reads a string from user; use int() or float() to convert. print() displays values.

name = input("Enter your name: ")
print("Hello,", name)

5. Comments

Use # for single-line comments and triple quotes for multi-line comments.

# single-line comment
'''
Multi-line comment
'''

Common pitfalls (short)

  • Floating-point precision ‚Äî compare floats with tolerance.
  • Mutable default arguments ‚Äî avoid using mutable objects as default function arguments.
  • Dict keys must be hashable ‚Äî mutable types (list, set, dict) cannot be keys.
  • Use is None to test for None.

Exercise 3 ‚Äî Check Positive or Negative (use boolean)

# Task: Ask user for a number and print whether it's positive, negative or zero

# Solution (example program):
def check_number(n):
    if n > 0:
        return True  # positive
    elif n < 0:
        return False # negative
    else:
        return None  # zero

# Example usage:
for test in [10, -5, 0]:
    result = check_number(test)
    if result is True:
        print(test, "is positive")
    elif result is False:
        print(test, "is negative")
    else:
        print(test, "is zero")

# Expected output:
# 10 is positive
# -5 is negative
# 0 is zero
Alternate simpler solution (direct print) 
n = int(input("Enter a number: "))
if n > 0:
    print("Positive")
elif n < 0:
    print("Negative")
else:
    print("Zero")
Tips:
  • Use type(x) and isinstance(x, T) to inspect data.
  • Remember that empty containers ("", [], {}, set()) are falsy in boolean contexts.
  • Convert strings to numbers carefully: int("3.14") fails ‚Äî do float() first.
