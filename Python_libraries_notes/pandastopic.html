Topic 1: Introduction to Pandas & Series

Pandas is a Python library for data manipulation and analysis. It provides two main data structures: Series (1D) and DataFrame (2D tabular data). Series can be created from lists, tuples, dictionaries, or NumPy arrays. DataFrames are ideal for structured data like CSV files or Excel sheets.

import pandas as pd

# From list
s1 = pd.Series([2,3,4])

# From dictionary
dic = {'id': 4, 'Name': "Anna", 'Place': "Eklm", 'Age': 45}
s2 = pd.Series(dic, index=['Name','Age','id','Place'])

# Using range
s3 = pd.Series([i for i in range(6)])
print(s1)
print(s2)
print(s3)

s = pd.Series([4,5,5,3,2,5,7])
print("Shape:", s.shape)
print("Size:", s.size)
print("Data type:", s.dtype)
print("Number of dimensions:", s.ndim)
print("First 5 elements:", s.head())

a = pd.Series([4,3,2])
b = pd.Series([1,2,3])

print("Addition:", a.add(b))
print("Subtraction:", a.sub(b))
print("Multiplication:", a.mul(b))
print("Division:", a.div(b))

sales = pd.Series([2500, 3000, 2800, 3500, 4000], 
                  index=['Jan','Feb','Mar','Apr','May'])

print("Months:", sales.index)
print("Values:", sales.values)
print("Sales +500:\n", sales + 500)

Q2. Difference between Series and DataFrame?

A: Series is 1D, DataFrame is 2D (rows and columns).

Q3. How to create a Series from a dictionary?

A: Use pd.Series(dictionary) and optionally a custom index.

Q4. How to perform element-wise arithmetic operations?

A: Use operators (+,-,*,/) or methods like add(), sub().

Q5. How to get the first few elements of a Series?

A: Use head(n) method; default is 5 elements.

Topic 2: DataFrame Basics & Operations

A DataFrame is a two-dimensional labeled data structure in Pandas. It can store data of different types (numeric, string, boolean) in columns. DataFrames are ideal for CSV, Excel, SQL tables, and structured datasets.

import pandas as pd

# From list of lists
data = [
    [101,'Vineeth',32,'Alappuzha'],
    [102,'Amal',25,'Ernakulam'],
    [103,'Aleena',21,'Kottayam']
]
df1 = pd.DataFrame(data, columns=['ID','Name','Age','Place'])

# From dictionary
dic = {
    'Roll':[101,102,103],
    'Name':['Amal','Anu','Raj'],
    'Age':[24,43,21],
    'Place':['Alapy','Kottayam','Kannur']
}
df2 = pd.DataFrame(dic)

print(df1)
print(df2)

print("Shape:", df1.shape)
print("Number of elements:", df1.size)
print("Data types:\n", df1.dtypes)
print("Column names:", df1.columns)
print("Number of dimensions:", df1.ndim)
print("First 5 rows:\n", df1.head())
print("Last 5 rows:\n", df1.tail())

# Adding columns
df1['Gender'] = ['M','M','F']
df1['Salary'] = [20000,30000,40000]

# Modifying columns
df1['Salary'] = df1['Salary'] + 5000

print(df1)

# Using column names
print(df1['Name'])
print(df1[['Name','Age']])

# Using iloc (integer index)
print(df1.iloc[0])       # first row
print(df1.iloc[0:2, 1:3]) # first 2 rows, columns 1 to 2

# Using loc (label-based)
print(df1.loc[df1['Age']>25, ['Name','Age']])

# Detect missing values
print(df1.isna().sum())

# Fill missing values
df1['Place'].fillna('Unknown', inplace=True)

# Drop rows with missing values
df1.dropna(inplace=True)

# Sorting
df1_sorted = df1.sort_values(by='Age', ascending=False)

# Filtering

Topic 3: Advanced DataFrame Operations, Grouping & Merging

Advanced DataFrame operations include grouping, aggregation, merging, joining, and handling duplicates. These are essential for analyzing large datasets efficiently.

Use groupby() to group data by a column and perform aggregation functions like count(), sum(), mean(), max(), min().

import pandas as pd

# Sample DataFrame
data = {
    'Name':['Alice','Bob','Charlie','Alice','Bob'],
    'Department':['HR','IT','IT','HR','IT'],
    'Salary':[50000,60000,55000,52000,62000],
    'Age':[25,30,28,26,31]
}
df = pd.DataFrame(data)

# Group by Department
grouped = df.groupby('Department')

# Aggregations
count_per_dept = grouped['Name'].count()
sum_salary = grouped['Salary'].sum()
max_age = grouped['Age'].max()

print("Count per department:\n", count_per_dept)
print("Total Salary per department:\n", sum_salary)
print("Max Age per department:\n", max_age)

# Filter departments with total salary > 110000
filtered = grouped['Salary'].sum()
filtered = filtered[filtered > 110000]
print(filtered)

# Detect duplicates
df.duplicated().sum()

# Drop duplicates
df_unique = df.drop_duplicates()

Combine DataFrames using merge() and specify how:

  • inner: Only matching rows
  • left: All rows from left, matching from right
  • right: All rows from right, matching from left
  • outer: All rows from both DataFrames
# Sample DataFrames
df1 = pd.DataFrame({
    'ID':[1,2,3],
    'Name':['Alice','Bob','Charlie']
})
df2 = pd.DataFrame({
    'ID':[2,3,4],
    'Salary':[60000,55000,50000]
})

# Inner join
inner_join = pd.merge(df1, df2, on='ID', how='inner')

# Left join
left_join = pd.merge(df1, df2, on='ID', how='left')

# Outer join
outer_join = pd.merge(df1, df2, on='ID', how='outer')

print("Inner Join:\n", inner_join)
print("Left Join:\n", left_join)
print("Outer Join:\n", outer_join)

Topic 5: Loc, Iloc & Advanced Indexing

Pandas provides powerful ways to select and filter data using loc (label-based) and iloc (integer-based) indexing. You can select rows, columns, or specific cells, and combine conditions to extract meaningful subsets.

Topic 6: GroupBy, Aggregations & Joins

Pandas provides GroupBy functionality to group data based on one or more columns and perform aggregations such as sum, mean, count, max, min. Additionally, Pandas supports joins/merges to combine multiple DataFrames using common columns.

import pandas as pd

# Sample DataFrame
data = {
    'ID':[101,102,103,104,105,106],
    'Name':['Alice','Bob','Charlie','Diana','Eve','Frank'],
    'Age':[25,30,28,32,28,30],
    'Profession':['Engineer','Doctor','Engineer','Nurse','Doctor','Nurse'],
    'Salary':[50000,70000,55000,60000,72000,58000]
}
df = pd.DataFrame(data)

# Group by Profession and count number of people
profession_count = df.groupby('Profession')['Name'].count()
print(profession_count)

# Group by Profession and calculate mean salary
mean_salary = df.groupby('Profession')['Salary'].mean()
print(mean_salary)

# Multiple aggregations
agg_df = df.groupby('Profession')['Salary'].agg(['mean','max','min'])
print(agg_df)

Pandas supports inner, left, right, and outer joins to combine two DataFrames based on a common column.

# DataFrames to merge
df1 = pd.DataFrame({
    'ID':[101,102,103],
    'Name':['Alice','Bob','Charlie']
})

df2 = pd.DataFrame({
    'ID':[102,103,104],
    'Salary':[70000,55000,60000]
})

# Inner join (only matching IDs)
inner_join = pd.merge(df1, df2, on='ID', how='inner')
print(inner_join)

# Left join (all rows from left + matching from right)
left_join = pd.merge(df1, df2, on='ID', how='left')
print(left_join)

# Right join (all rows from right + matching from left)
right_join = pd.merge(df1, df2, on='ID', how='right')
print(right_join)

# Outer join (all rows from both)
outer_join = pd.merge(df1, df2, on='ID', how='outer')
print(outer_join)

Q1. What is GroupBy in Pandas?

A: GroupBy allows grouping data by one or more columns and applying aggregation functions.

Q2. What aggregations can be performed after GroupBy?

A: Common aggregations: count, sum, mean, min, max, agg() for multiple functions.

Q3. Difference between inner, left, right, and outer joins?

A:

  • Inner: only matching rows
  • Left: all rows from left + matching from right
  • Right: all rows from right + matching from left
  • Outer: all rows from both

Q4. How to merge DataFrames on multiple columns?

A: Use pd.merge(df1, df2, on=['col1','col2'], how='type').

Q5. Can you perform aggregation on multiple columns simultaneously?

A: Yes, using agg() with a dictionary: df.groupby('col').agg({'col1':'sum','col2':'mean'}).

Topic 7: Handling Missing Data, Data Cleaning & Correcting Wrong Data

Data in real-world datasets often contains missing or incorrect values. Pandas provides methods to detect, fill, or drop missing values and correct wrong data efficiently.

import pandas as pd

df = pd.read_csv('customer.csv')  # Sample CSV
print(df.isna())          # Boolean DataFrame: True if missing
print(df.isna().sum())    # Count of missing values per column

Use fillna() to replace missing values with a constant, mean, median, or mode.

# Fill all missing values with a constant
df.fillna('India', inplace=True)

# Fill numeric column with mean
df['age'].fillna(df['age'].mean(), inplace=True)

# Fill with mode
df['place'].fillna(df['place'].mode()[0], inplace=True)

Use dropna() to remove rows or columns with missing values.

# Drop rows with any missing value
df.dropna(inplace=True)

# Drop specific column if it has missing values
df.drop(['column_name'], axis=1, inplace=True)

Use loc to locate incorrect entries and replace them.

# Suppose 'Iname' at index 460 is wrong
df.loc[460, 'Iname'] = 'John'

# Example: Replace all negative ages with mean age
df.loc[df['age'] < 0, 'age'] = df['age'].mean()

Q1. How to check for missing values in a DataFrame?

A: Use df.isna() for a boolean DataFrame or df.isna().sum() to count missing per column.

Q2. Difference between fillna() and dropna()?

A: fillna() replaces missing values; dropna() removes rows/columns with missing values.

Q3. How to fill missing values with column mean, median, or mode?

A: df['col'].fillna(df['col'].mean(), inplace=True) or median() / mode()[0].

Q4. How to correct wrong data in Pandas?

A: Use df.loc[index, 'column'] = new_value or conditional replacement like df.loc[df['col'] < 0, 'col'] = value.

Q5. Can fillna() be used for multiple columns simultaneously?

A: Yes, using a dictionary: df.fillna({'col1':value1, 'col2':value2}, inplace=True).

Topic 8: Sorting, Filtering & Selecting Data

Sorting and filtering are essential for analyzing large datasets. Pandas provides methods to sort rows or columns, select specific rows/columns, and filter data based on conditions.

Pandas allows easy selection of specific columns or rows using loc, iloc, or simple indexing.

import pandas as pd

# Sample dataset
df = pd.read_csv('customer.csv', header=None)
df.columns = ['id', 'fname', 'Iname', 'age', 'profession', 'place']

# Select single column
print(df['fname'])

# Select multiple columns
print(df[['fname', 'age', 'place']])

# Select rows using iloc (integer index)
print(df.iloc[0:5])  # First 5 rows

# Select rows using loc (label index)
print(df.loc[0:5, ['fname', 'age']])  # First 6 rows, fname and age columns

Filter rows based on specific conditions using boolean indexing.

# Age above 60
df_age = df.loc[df['age'] > 60]
print(df_age)

# Place is 'India'
df_place = df.loc[df['place'] == 'India']
print(df_place)

# Age between 50 and 60 AND place is 'Africa'
df_filtered = df.loc[(df['age'] > 50) & (df['age'] <= 60) & (df['place'] == 'Africa')]
print(df_filtered)

# Profession is 'Photographer'
df_prof = df.loc[df['profession'] == 'Photographer']
print(df_prof)

You can sort a DataFrame by one or more columns using sort_values(). Sorting can be ascending or descending.

# Sort by age (ascending)
df_sorted = df.sort_values(by='age')
print(df_sorted)

# Sort by place (descending)
df_sorted_desc = df.sort_values(by='place', ascending=False)
print(df_sorted_desc)

# Get top 2 people with maximum age in 'Africa'
df_top_age = df.loc[df['place'] == 'Africa'].sort_values(by='age', ascending=False).head(2)
print(df_top_age)

# Get 4 people with minimum age whose profession is 'Nurse'
df_min_age_nurse = df.loc[df['profession'] == 'Nurse'].sort_values(by='age').head(4)
print(df_min_age_nurse)

Remove unwanted rows or columns using drop().

# Drop a column
df_drop_col = df.drop(['Iname'], axis=1)
print(df_drop_col)

# Drop a row by index
df_drop_row = df.drop([460], axis=0)
print(df_drop_row)

Topic 10: Merging, Joining & Concatenating DataFrames

In Pandas, combining datasets is essential for data analysis. You can merge, join, or concatenate DataFrames based on columns or indexes.

  • Merge: Combines DataFrames based on common columns or indices (similar to SQL joins).
  • Join: Convenient method for combining DataFrames using index or key columns.
  • Concatenate: Stacks DataFrames vertically or horizontally.

Use pd.merge() to combine DataFrames on a common column.

import pandas as pd

# Sample DataFrames
df1 = pd.DataFrame({
    'id': [101, 102, 103],
    'Name': ['Alice', 'Bob', 'Charlie']
})

df2 = pd.DataFrame({
    'id': [101, 102, 104],
    'Salary': [50000, 60000, 70000]
})

# Inner join (only matching ids)
inner_df = pd.merge(df1, df2, on='id', how='inner')
print("Inner Join:\n", inner_df)

# Left join (all rows from left + matching from right)
left_df = pd.merge(df1, df2, on='id', how='left')
print("Left Join:\n", left_df)

# Right join
right_df = pd.merge(df1, df2, on='id', how='right')
print("Right Join:\n", right_df)

# Full outer join
outer_df = pd.merge(df1, df2, on='id', how='outer')
print("Outer Join:\n", outer_df)

You can join DataFrames based on index instead of columns:

df1 = pd.DataFrame({'Name': ['Alice', 'Bob', 'Charlie']}, index=[101, 102, 103])
df2 = pd.DataFrame({'Salary': [50000, 60000, 70000]}, index=[101, 102, 104])

# Join using index
joined_df = df1.join(df2, how='outer')
print(joined_df)

Stack DataFrames vertically or horizontally using pd.concat().

df1 = pd.DataFrame({'Name': ['Alice', 'Bob'], 'Age': [25, 30]})
df2 = pd.DataFrame({'Name': ['Charlie', 'David'], 'Age': [35, 40]})

# Vertical concatenation (stack rows)
vert_concat = pd.concat([df1, df2])
print("Vertical Concatenation:\n", vert_concat)

# Horizontal concatenation (stack columns)
df3 = pd.DataFrame({'Salary': [50000, 60000]})
horiz_concat = pd.concat([df1, df3], axis=1)
print("Horizontal Concatenation:\n", horiz_concat)

Q1. What is the difference between merge and join?

A: merge() is SQL-style, used to combine DataFrames on columns. join() is a convenient method to join on indexes or key columns.

Q2. Difference between inner, left, right, and outer joins?

A: Inner: only matching rows. Left: all rows from left + matching from right. Right: all from right + matching from left. Outer: all rows from both, filling missing with NaN.

Q3. How to concatenate DataFrames vertically and horizontally?

A: Use pd.concat([df1, df2]) for vertical (default axis=0) and pd.concat([df1, df2], axis=1) for horizontal.

Q4. Can you join DataFrames on index?

A: Yes, df1.join(df2, how='outer') joins DataFrames on index.

Q5. When to use merge vs concatenation?

A: Use merge when combining datasets based on key columns (like relational tables). Use concat when stacking datasets vertically or horizontally without key columns.
