Topic 1: Introduction to Pandas & Series

Pandas is a Python library for data manipulation and analysis. It provides two main data structures: Series (1D) and DataFrame (2D tabular data). Series can be created from lists, tuples, dictionaries, or NumPy arrays. DataFrames are ideal for structured data like CSV files or Excel sheets.

import pandas as pd

# From list
s1 = pd.Series([2,3,4])

# From dictionary
dic = {'id': 4, 'Name': "Anna", 'Place': "Eklm", 'Age': 45}
s2 = pd.Series(dic, index=['Name','Age','id','Place'])

# Using range
s3 = pd.Series([i for i in range(6)])
print(s1)
print(s2)
print(s3)

s = pd.Series([4,5,5,3,2,5,7])
print("Shape:", s.shape)
print("Size:", s.size)
print("Data type:", s.dtype)
print("Number of dimensions:", s.ndim)
print("First 5 elements:", s.head())

a = pd.Series([4,3,2])
b = pd.Series([1,2,3])

print("Addition:", a.add(b))
print("Subtraction:", a.sub(b))
print("Multiplication:", a.mul(b))
print("Division:", a.div(b))

sales = pd.Series([2500, 3000, 2800, 3500, 4000], 
                  index=['Jan','Feb','Mar','Apr','May'])

print("Months:", sales.index)
print("Values:", sales.values)
print("Sales +500:\n", sales + 500)

Q1. What is a Pandas Series?

A: A Series is a one-dimensional labeled array capable of holding any data type.

Q2. Difference between Series and DataFrame?

A: Series is 1D, DataFrame is 2D (rows and columns).

Q3. How to create a Series from a dictionary?

A: Use pd.Series(dictionary) and optionally a custom index.

Q4. How to perform element-wise arithmetic operations?

A: Use operators (+,-,*,/) or methods like add(), sub().

Q5. How to get the first few elements of a Series?

A: Use head(n) method; default is 5 elements.

Topic 2: DataFrame Basics & Operations

A DataFrame is a two-dimensional labeled data structure in Pandas. It can store data of different types (numeric, string, boolean) in columns. DataFrames are ideal for CSV, Excel, SQL tables, and structured datasets.

import pandas as pd

# From list of lists
data = [
    [101,'Vineeth',32,'Alappuzha'],
    [102,'Amal',25,'Ernakulam'],
    [103,'Aleena',21,'Kottayam']
]
df1 = pd.DataFrame(data, columns=['ID','Name','Age','Place'])

# From dictionary
dic = {
    'Roll':[101,102,103],
    'Name':['Amal','Anu','Raj'],
    'Age':[24,43,21],
    'Place':['Alapy','Kottayam','Kannur']
}
df2 = pd.DataFrame(dic)

print(df1)
print(df2)

print("Shape:", df1.shape)
print("Number of elements:", df1.size)
print("Data types:\n", df1.dtypes)
print("Column names:", df1.columns)
print("Number of dimensions:", df1.ndim)
print("First 5 rows:\n", df1.head())
print("Last 5 rows:\n", df1.tail())

# Adding columns
df1['Gender'] = ['M','M','F']
df1['Salary'] = [20000,30000,40000]

# Modifying columns
df1['Salary'] = df1['Salary'] + 5000

print(df1)

# Using column names
print(df1['Name'])
print(df1[['Name','Age']])

# Using iloc (integer index)
print(df1.iloc[0])       # first row
print(df1.iloc[0:2, 1:3]) # first 2 rows, columns 1 to 2

# Using loc (label-based)
print(df1.loc[df1['Age']>25, ['Name','Age']])

# Detect missing values
print(df1.isna().sum())

# Fill missing values
df1['Place'].fillna('Unknown', inplace=True)

# Drop rows with missing values
df1.dropna(inplace=True)

# Sorting
df1_sorted = df1.sort_values(by='Age', ascending=False)

# Filtering
df_filtered = df1.loc[(df1['Age']>20) & (df1['Place']=='Alappuzha')]

print(df1_sorted)
print(df_filtered)

Topic 3: Advanced DataFrame Operations, Grouping & Merging

Advanced DataFrame operations include grouping, aggregation, merging, joining, and handling duplicates. These are essential for analyzing large datasets efficiently.

Use groupby() to group data by a column and perform aggregation functions like count(), sum(), mean(), max(), min().

import pandas as pd

# Sample DataFrame
data = {
    'Name':['Alice','Bob','Charlie','Alice','Bob'],
    'Department':['HR','IT','IT','HR','IT'],
    'Salary':[50000,60000,55000,52000,62000],
    'Age':[25,30,28,26,31]
}
df = pd.DataFrame(data)

# Group by Department
grouped = df.groupby('Department')

# Aggregations
count_per_dept = grouped['Name'].count()
sum_salary = grouped['Salary'].sum()
max_age = grouped['Age'].max()

print("Count per department:\n", count_per_dept)
print("Total Salary per department:\n", sum_salary)
print("Max Age per department:\n", max_age)

# Filter departments with total salary > 110000
filtered = grouped['Salary'].sum()
filtered = filtered[filtered > 110000]
print(filtered)

# Detect duplicates
df.duplicated().sum()

# Drop duplicates
df_unique = df.drop_duplicates()

Combine DataFrames using merge() and specify how:

  • inner: Only matching rows
  • left: All rows from left, matching from right
  • right: All rows from right, matching from left
  • outer: All rows from both DataFrames
# Sample DataFrames
df1 = pd.DataFrame({
    'ID':[1,2,3],
    'Name':['Alice','Bob','Charlie']
})
df2 = pd.DataFrame({
    'ID':[2,3,4],
    'Salary':[60000,55000,50000]
})

# Inner join
inner_join = pd.merge(df1, df2, on='ID', how='inner')

# Left join
left_join = pd.merge(df1, df2, on='ID', how='left')

# Outer join
outer_join = pd.merge(df1, df2, on='ID', how='outer')

print("Inner Join:\n", inner_join)
print("Left Join:\n", left_join)
print("Outer Join:\n", outer_join)

Q1. What is the purpose of groupby in Pandas?

A: To split the DataFrame into groups based on column(s) and perform aggregate computations like sum, mean, count, etc.

Q2. Difference between inner and outer merge?

A: Inner merge keeps only matching rows; outer merge keeps all rows from both DataFrames with NaN for missing values.

Q3. How to handle duplicates in a DataFrame?

A: Use drop_duplicates() to remove duplicate rows; use duplicated() to detect duplicates.

Q4. How to filter grouped data?

A: After groupby(), perform aggregation and then apply boolean indexing to filter rows.

Q5. What are common join types in Pandas merge?

A: inner, left, right, outer — depending on which rows you want to keep from each DataFrame.

Topic 4: Data Cleaning, Handling Missing/Wrong Data & Advanced Filters

Data cleaning is crucial for reliable analysis. Key steps include handling missing values, correcting wrong data, filtering rows based on conditions, and renaming columns. Pandas provides isna(), fillna(), dropna(), loc, iloc, and boolean indexing for these tasks.

import pandas as pd

# Sample DataFrame with missing values
data = {
    'ID':[101,102,103,104],
    'Name':['Alice','Bob',None,'Diana'],
    'Age':[25,None,28,30],
    'Place':['NY','LA','NY',None]
}
df = pd.DataFrame(data)

# Detect missing values
print(df.isna())

# Fill missing values with a single value
df.fillna({'Name':'Unknown', 'Age':df['Age'].mean(), 'Place':'Unknown'}, inplace=True)

# Drop rows with any missing values
df.dropna(inplace=True)

# Correct wrong data using loc
df.loc[df['Name']=='Alicee','Name'] = 'Alice'
df.loc[df['Place']=='Nw York','Place'] = 'NY'

# Extract specific columns
print(df[['Name','Place']])

# Filter rows where Age > 25
df_age = df.loc[df['Age'] > 25]

# Filter rows where Place is 'NY' and Age > 25
df_filtered = df.loc[(df['Place']=='NY') & (df['Age']>25)]

# Rename columns using rename()
df.rename(columns={'ID':'CustomerID','Name':'FullName'}, inplace=True)
print(df.columns)

# Sort by Age ascending
df_sorted = df.sort_values(by='Age')

# Sort by Place descending
df_sorted_desc = df.sort_values(by='Place', ascending=False)

Q1. How do you detect missing values in Pandas?

A: Use isna() or isnull() to detect missing values in DataFrame.

Q2. How can you fill missing values?

A: Use fillna() to replace missing values with a specific value, mean, median, or mode.

Q3. How to remove rows with missing values?

A: Use dropna() to remove rows that have any NaN values.

Q4. How do you correct wrong data in a DataFrame?

A: Use loc[index, 'column'] = new_value or boolean indexing to fix incorrect entries.

Q5. How do you filter data based on multiple conditions?

A: Use boolean indexing with & (and), | (or), and parentheses: df.loc[(cond1) & (cond2)].

Topic 5: Loc, Iloc & Advanced Indexing

Pandas provides powerful ways to select and filter data using loc (label-based) and iloc (integer-based) indexing. You can select rows, columns, or specific cells, and combine conditions to extract meaningful subsets.

import pandas as pd

# Sample DataFrame
data = {
    'ID':[101,102,103,104],
    'Name':['Alice','Bob','Charlie','Diana'],
    'Age':[25,30,28,32],
    'Place':['NY','LA','SF','NY']
}
df = pd.DataFrame(data)

# Select single row by label (index)
print(df.loc[1])

# Select multiple rows by labels
print(df.loc[[0,2]])

# Select specific row and column
print(df.loc[2, 'Name'])

# Select multiple rows and columns
print(df.loc[[0,2], ['Name','Place']])

# Select row by integer position
print(df.iloc[1])

# Select multiple rows
print(df.iloc[[0,2]])

# Select specific row and column
print(df.iloc[2,1])

# Select multiple rows and columns
print(df.iloc[0:3, 1:3])

# Filter rows where Age > 28
df_age = df.loc[df['Age'] > 28]

# Filter rows with multiple conditions (Age>25 and Place='NY')
df_filtered = df.loc[(df['Age']>25) & (df['Place']=='NY')]

# Select specific columns after filtering
df_select = df_filtered[['Name','Place','Age']]

# Single column
names = df['Name']

# Multiple columns
subset = df[['Name','Age']]

# Using loc for columns
subset2 = df.loc[:, ['ID','Place']]

Q1. Difference between loc and iloc?

A: loc is label-based; iloc is integer-position based.

Q2. How do you select multiple rows and columns?

A: Use df.loc[[row1,row2], [col1,col2]] or df.iloc[[0,2],[1,3]].

Q3. How do you filter data using multiple conditions?

A: Combine boolean conditions with & (and), | (or), and parentheses: df.loc[(cond1)&(cond2)].

Q4. How to select a single column?

A: Use df['ColumnName'] or df.loc[:, 'ColumnName'].

Q5. How to select multiple columns?

A: Use df[['Col1','Col2']] or df.loc[:, ['Col1','Col2']].
