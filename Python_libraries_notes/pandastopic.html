Topic 1: Introduction to Pandas & Series

Pandas is a powerful Python library used for data manipulation and analysis. It provides two primary data structures: Series (1D) and DataFrame (2D tabular data). Series can be created from lists, tuples, dictionaries, or NumPy arrays. DataFrames are ideal for representing structured data like CSV files or Excel sheets.

Series are one-dimensional labeled arrays that can hold any data type.

import pandas as pd

# From list
list1 = [2, 3, 4]
s1 = pd.Series(list1)
print(s1)

# From tuple
tup1 = (10, 20, 30)
s2 = pd.Series(tup1)

# From dictionary
dic = {'id': 4, 'Name': "Anna", 'Place': "Eklm", 'Age': 45}
s3 = pd.Series(dic, index=['Name', 'Age', 'id', 'Place'])

# Using range
s4 = pd.Series([i for i in range(6)])

Useful attributes to inspect a Series:

s = pd.Series([4, 5, 5, 3, 2, 5, 7])
print("Shape:", s.shape)
print("Size:", s.size)
print("Data type:", s.dtype)
print("Number of dimensions:", s.ndim)
print("First 5 elements:", s.head())

Perform element-wise arithmetic operations:

a = pd.Series([4, 3, 2])
b = pd.Series([1, 2, 3])

c_add = a.add(b)      # Addition
c_sub = a.sub(b)      # Subtraction
c_mul = a.mul(b)      # Multiplication
c_div = a.div(b)      # Division

print("Addition:", c_add)
print("Subtraction:", c_sub)
print("Multiplication:", c_mul)
print("Division:", c_div)

Exercise 1 – Series Addition

s1 = pd.Series([5, 10, 15])

print("Addition:", s1 + s2)

Coding Challenge – Temperature Data

temps = pd.Series([22, 25, 21, 23, 24, 26, 22])
print("Max temp:", temps.max())
print("Min temp:", temps.min())
print("Mean temp:", temps.mean())

Creating a Series of monthly sales and performing basic operations:

sales = pd.Series([2500, 3000, 2800, 3500, 4000], 
                  index=['Jan','Feb','Mar','Apr','May'])

# Basic attributes
print("Total months:", sales.size)
print("Months:", sales.index)
print("Values:", sales.values)

# Arithmetic operations
increase = sales + 500
print("Sales after increase:\n", increase)

Q1. What is a Pandas Series?

A: A Series is a one-dimensional labeled array capable of holding any data type.

Q2. Difference between Series and DataFrame?

A: Series is 1D, DataFrame is 2D (rows and columns).

Q3. How to create a Series from a dictionary?

A: Use pd.Series(dictionary). You can also specify custom index labels.

Q4. How to perform element-wise arithmetic operations on Series?

A: Use operators (`+`, `-`, `*`, `/`) or methods like add(), sub().

Q5. How to get the first few elements of a Series?

A: Use head(n) method; default is 5 elements.
